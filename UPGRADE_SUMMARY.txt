================================================================================
  HOW TO USE VERSION 2.0
================================================================================

BASIC USAGE (Same as before):
  1. Open flashcard.html in your browser
  2. Upload any CSV file (German,English format)
  3. Start learning!

NEW FEATURES:

A) FLEXIBLE COLUMN ORDER:
   Your CSV can be:
   - German,English     ‚úÖ
   - English,German     ‚úÖ NEW! Auto-detected
   - With headers       ‚úÖ NEW! Auto-skipped

B) EXTENDED CONTENT:
   Create CSV like this:
   Word, Example Sentence, Word, Example Sentence

   Example:
   Hallo,Ich sage hallo zu Freunden,Hello,I say hello to friends

C) ENABLE EXTENDED READING:
   1. Upload extended CSV
   2. Open: Settings ‚Üí Display Options
   3. Check: "üìñ Read Extended Examples/Sentences in Auto-Play"
   4. Start Auto-Play

   Now it reads full sentences instead of just words!

Q: Wrong language detection?
A: Add German articles (der, die, das) or use German characters (√§, √∂, √º)

Q: Old CSV not working?
A: Should work fine! Check file encoding is UTF-8

A: Use flashcard_v1.0_backup.html

Q: CSV format errors?
